1. Задание:
Разработать программу для анализа данных с температурных датчиков в разных городах. Данные представляют собой двумерный массив (до 30 дней по 24 измерения каждый). Программа должна:
•	Запросить количество дней и измерений в дне
•	Создать двумерный динамический массив
•	Считать значения температуры с клавиатуры
•	Реализовать функции:
o	dayAverage() — средняя температура за день
o	overallAverage() — общая средняя температура
o	showHotDays() — вывод дней, где средняя температура дня выше общей средней
•	Освободить память

2. Описание алгоритма:
•	Использование указателя на указатель (float**) для создания двумерного динамического массива
•	Ввод количества дней и измерений с проверкой на максимальные значения (30 дней и 24 измерения)
•	Двойное выделение памяти: сначала массив указателей, затем для каждого дня — массив значений
•	Ввод данных с использованием арифметики указателей *(*(data+i)+j)
•	Функция dayAverage(): вычисляет среднюю температуру для одного дня через суммирование элементов строки
•	Функция overallAverage(): вычисляет общую среднюю температуру как сумму всех измерений за все дни, делённую на общее количество измерений (d*h). 
•	Функция showHotDays(): находит дни, где средняя температура превышает общую среднюю
•	Освобождение памяти в обратном порядке: сначала массивы измерений, затем массив указателей
